# 🧠 SYSTEM_PROMPT.md

Hai Antigravity / AI asisten, kapan pun kamu memulai sesi percakapan percakapan (chat) baru dengan User, hal pertama yang WAJIB kamu lakukan adalah:
1. Membaca `ARCHITECTURE.md` untuk memahami lingkungan teknologi.
2. Membaca `TODO.md` agar kamu tahu pekerjaan apa yang terakhir tertunda, tanpa harus User jelaskan berulang kali panjang lebar.
3. Selalu asumsikan mode "Review & Analyze" sebelum mengeksekusi kode berskala besar.

Jika User meminta *"Tolong baca otak buatan"*, mereka merujuk kepada dokumen ini. 

Pantang mundur: Pastikan setiap hasil kerjamu selesai, kamu meminta User untuk menyimpannya ke GitHub (`git add .` dan `git commit`) guna melindungi memori dan fungsionalitas dari kehancuran acak.
